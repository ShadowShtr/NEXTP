# assets/illustrations/png

Ilustrações 3D oficiais do NextP (pack `nextp_all_png_3d_assets`), fonte original em alta resolução. As versões otimizadas para a web (redimensionadas, <130KB) estão em `public/illustrations/` e usadas diretamente nos headers das abas:

- `registos-mascot.png` — mascote com telemóvel (aba Registos)
- `guardados-wallet.png` — carteira (aba Guardados)
- `planeamento-piggybank.png` — cofrinho (aba Planeamento)
- `resumo-wallet-growth.png` — carteira com gráfico de crescimento (aba Resumo)
- `finance-*.png` — ilustrações extra (gráficos, cartão, cofrinho+moedas, carteira+moeda), disponíveis para uso futuro.

O ícone oficial do app está em `assets/icons/nextp-app-icon-master.png`, usado para gerar `public/icons/icon-*.png` e `maskable-512.png`.

# assets/icons

Ficheiros do ícone/logo oficial do NextP. Ver `docs/12-logo-e-icone.md`.

Colocar aqui o **master 1024×1024** e os tamanhos derivados:

```
nextp-icon-1024.png   nextp-icon-512.png   nextp-icon-192.png
nextp-icon-144.png    nextp-icon-96.png    nextp-icon-72.png    nextp-icon-48.png
```

> O ícone que já funciona no APK está recriado como vetor em
> `app/src/main/res/drawable/ic_launcher_foreground.xml` (+ adaptativo em `mipmap-anydpi-v26`).
